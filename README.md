# TareaSemana14-Crud-Modular



\-- Sistema de Registro de Visitantes (CRUD con Tkinter) --



Aplicación de escritorio desarrollada en Python utilizando Tkinter para la gestión de visitantes en una oficina.



El sistema permite realizar operaciones CRUD (Crear, Leer, Eliminar) sobre los registros de visitantes, aplicando una arquitectura modular por capas:

\- Modelos → Representación de los datos (Visitante).

\- Servicios → Lógica de negocio y operaciones CRUD.

\- UI → Interfaz gráfica con Tkinter.

\- Main → Punto de entrada de la aplicación.
